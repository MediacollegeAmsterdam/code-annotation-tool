# Vragenlijst voor Studenten - Code Annotatie Tool

## Introductie
Deze vragen helpen ons begrijpen hoe de Code Annotatie Tool jou helpt bij het leren programmeren. Voel je vrij om eerlijk te zijn - er zijn geen verkeerde antwoorden!

---

## A. Leren en Begrip

### 1. Code Begrip
Heeft de tool je geholpen om code beter te begrijpen? Kun je een voorbeeld geven van wanneer een uitleg iets duidelijk maakte?

### 2. Visuele Verbindingen
Helpen de pijlen die code met uitleg verbinden om te zien welk stuk bij welke uitleg hoort? Waarom wel/niet?

### 3. Concepten Onthouden
Denk je dat je code concepten makkelijker onthoudt wanneer je ze met de tool hebt bekeken vergeleken met gewone tekst uitleg?

### 4. Moeilijke Onderwerpen
Heb je de tool gebruikt voor moeilijke programmeer concepten? Was het nuttig? Wat maakte het nuttig of juist niet?

### 5. Eigen Tempo
Helpt het dat je zelf door de uitleg slides kunt navigeren (eigen tempo)? Waarom is dit wel/niet handig?

## C. Motivatie en Betrokkenheid

### 11. Interessant vs Saai
Vind je het interessanter om code te leren met deze tool vergeleken met anders? Waarom?

### 12. Vaker Gebruiken
Zou je deze tool vaker willen gebruiken bij het leren? Waarom wel/niet?

### 13. Delen met Anderen
Zou je deze tool aanraden aan medestudenten? Waarom wel/niet?

### 14. Leuk om te Gebruiken
Is de tool "leuk" om te gebruiken of voelt het als extra werk? Leg uit.

### 15. Nieuwsgierigheid

## D. Praktisch Gebruik

### 16. Wanneer Gebruik Je Het?
In welke situaties gebruik je (of zou je gebruiken) de tool? Bijvoorbeeld:
- Bij het leren van nieuwe syntax?
- Bij het debuggen?
- Bij het bestuderen van voorbeelden?
- Anders?

### 17. Integratie met Leren
Hoe past de tool in je manier van leren? Gebruik je het samen met andere bronnen (YouTube, documentatie, lessen)?

### 18. Oefenen met Code
Helpt de tool je als je zelf code schrijft en oefent? Hoe?

### 19. Samenwerken
Heb je de tool gebruikt samen met andere studenten (bijvoorbeeld om samen code te bekijken)? Hoe werkte dat?

### 20. Les Materiaal
Als je docent code uitleg met de tool zou delen, zou dat handig zijn? Waarom?

### 23. Ontbrekende Features
